# Koshu.AssemblyInfo

Plugin for generating assembly info files.

## Usage

    Generate-AssemblyInfo `
        -file ".\SharedAssemblyInfo.cs" `
        -description "Description" `
        -company "Company name" `
        -product "Product name" `
        -version "1.0.0" `
        -buildVersion "1.0.0.0" `
        -copyright "2014, Company name" `
        -clsCompliant "false"
